# Tempo — Temporary Group Maker

A focused messaging app where every group has an explicit deadline. Create time-bound conversations, track countdowns, and handle expiry with purpose: close, extend, branch into sub-groups, or archive.

## Stack
- **Frontend:** React + Vite + TanStack Query
- **Backend:** Express 5 + Drizzle ORM
- **Database:** PostgreSQL
- **API:** Contract-first OpenAPI with Zod validation

## Features
- Multi-step group creation wizard
- Real-time urgency countdown timers (green → yellow → red)
- AI-generated summaries before group expiry
- Branch groups from expired conversations
- Dashboard with expiring-soon alerts and activity feed
